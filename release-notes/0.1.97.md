## 0.1.97

A reliability pass for installing CaveCMS on cPanel shared hosting.

### Fixed — your site starts correctly on cPanel

On cPanel's "Setup Node.js App", a freshly installed site could fail to start with a Node module error. It now boots reliably the moment you restart the app.

### Improved — a cleaner cPanel install

The installer no longer stops when cPanel has already created the app folder. It clears the placeholder starter files for you and carries on. And the "finish in cPanel" steps are now one short, correctly-ordered checklist instead of two overlapping lists.
