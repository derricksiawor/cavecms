## 0.1.61

The 0.1.60 reliability fix, now reachable in a single update from any recent
version.

CaveCMS installs, updates, and runs cleanly on every supported MySQL and
MariaDB version — including the MariaDB 10.6 that ships with Ubuntu 22.04 LTS.
0.1.60 introduced that fix; 0.1.61 lowers the update floor so an install a few
versions behind can pick it up directly from **Settings → Updates** instead of
stepping through each release in between.

Nothing to configure — updating applies everything automatically.
