## 0.1.76

The Media library now works on macOS installs out of the box.

If you run CaveCMS on a Mac (Apple Silicon or Intel) as a local or desktop
install, the admin Media library could fail to load with a server error —
the release previously bundled image-processing binaries for Linux only, so
the picker showed "We couldn't load your files." This release bundles the
macOS binaries too, so browsing and uploading media works everywhere. Hosted
Linux installs are unaffected.

Updating applies everything automatically.
