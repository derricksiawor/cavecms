## 0.1.99

Reinstalling on cPanel now just works, even after a failed attempt.

### Fixed — reinstalls no longer trip over leftover files

If an earlier install failed, its leftover build files (including hidden ones like `.next` and `.cavecms-state` that a manual `rm -rf *` skips and `ls` doesn't show) no longer block a fresh install. The installer recognises its own scaffolding and clears it for you. It still refuses to touch a live install's data (your sealed env file and your uploads) or any files it didn't create, and when it does stop it now lists exactly what's in the folder so you know what to clear.
