## 0.3.1

A clean, stable release that brings together the install and update hardening from the recent 0.2.x line. If your site is on an earlier version, this is the one to move to.

### Updates are dependable on shared hosting

Updates now run start to finish on cPanel and similar hosting. A bug in 0.2.6 through 0.2.8 could leave the updater unable to start at all; that is fixed. And if an update ever can't reach your site to verify it, it now tells you the exact address it tried and how to fix it, instead of a vague "not responding".

### Your Site URL can't quietly be wrong

Setup no longer lets a live site save a local address like localhost as its Site URL. A wrong Site URL used to silently break updates, your sitemap, and the links in your emails. If an existing site is set to localhost, the Updates page now flags it with a one-click link to fix it under Settings, General.
