## 0.1.62

Fixes the one-click "Update now" button.

The dashboard updater (Settings → Updates, and the update banner) couldn't
start an update — it left out a verification detail the server requires before
installing a download, so the button silently failed. Both now send it, and
"Update now" works again. The terminal `cavecms update` command was unaffected.

Updating applies everything automatically.
