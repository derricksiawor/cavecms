## 0.1.92

Two quality-of-life upgrades to the editor: a drag-and-drop **page outline** for restructuring layouts in seconds, and friendlier messages everywhere so you never see a raw error code again.

### New — restructure your page from a drag-and-drop outline

The editor's **Outline** panel is now fully interactive. See your whole page as a clean tree — every section, column, and block — and **drag to reorder, nest, or reparent** anything: drop a block into a different column, pull a card out to the top level, move a whole section, all with a live drag preview so you can see exactly where it lands. The outline and the page stay perfectly in sync — select a block on the page and it lights up in the outline, and the other way round. It's the fastest way to rework a layout without endless scrolling, and undo/redo stays reliable even after big structural moves.

### Improved — clear messages, never a raw error code

Across the dashboard — pages, posts, projects, media, leads, and the block editor — anything that used to occasionally surface a raw error code or status number now shows a plain-language message that tells you what happened and what to do next. If a block can't be dropped where you aimed, if a save needs a quick refresh, or if something briefly hiccups, you get a human sentence instead of a cryptic code.
