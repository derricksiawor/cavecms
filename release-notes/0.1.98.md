## 0.1.98

A follow-up to 0.1.97's cPanel install fixes.

### Fixed — cPanel installs no longer stall on leftover starter files

The installer now clears cPanel's placeholder scaffolding even when only part of it remains (for example, a lone `tmp/` folder that cPanel recreates after a failed run), so a reinstall just works instead of stopping with "directory not empty". Any real file in the folder still halts the install untouched.
