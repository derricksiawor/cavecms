# 0.1.82 — 22 new content blocks + a tidier block library

This release roughly doubles the block library — 22 new content and
marketing blocks — and reorganizes the editor's block picker into clear
categories so the growing library stays easy to scan. Every new block is
fully CMS-native: insert it from the palette, edit it in the drawer,
restyle, reorder, and clone it like any other block. No database
migration is required.

## New blocks

**Media & galleries**

- **Carousel** — a swipeable image slider with autoplay, looping,
  captions, and an optional link per slide.
- **Before / after** — a drag-to-compare image slider.
- **Hotspot** — an image with clickable tooltip markers you place
  anywhere on it.
- **Marquee** — a seamless scrolling logo strip or text ticker.

**Social proof**

- **Testimonial carousel** — rotating customer pull-quotes.
- **Reviews** — a card grid of star-rated reviews with author and avatar.
- **Star rating** — a star meter that supports half-stars.

**Marketing & pricing**

- **Pricing table** — a plan card with a price, a feature list, a button,
  and an optional "featured" highlight (compose three across a row).
- **Price list** — menu-style rows with a price beside each item.
- **Share buttons** — share the current page to X, LinkedIn, Facebook,
  email, or copy the link.

**Content & interactive**

- **Animated headline** — a heading with rotating or typewriter words.
- **Countdown** — a live countdown to any date and time.
- **Flip box** — a card that flips to reveal a back face on hover or tap.
- **Progress bars** — labeled skill/progress meters that fill as they
  scroll into view.
- **Progress tracker** — a numbered stepper / process indicator.
- **Timeline** — a dated vertical sequence of milestones.
- **Comparison table** — a feature matrix comparing up to four plans.
- **Toggle** — a single show/hide panel.

**Layout & navigation**

- **Table of contents** — jump links to anchored sections on the page.
- **Menu anchor** — an invisible in-page jump target for nav links.

**Embed & dynamic**

- **Embed** — embed YouTube, Vimeo, Spotify, CodePen, SoundCloud, or
  CodeSandbox from a normal share URL.
- **Code** — a code block with a filename, a language label, and optional
  line numbers.
- **Posts** — an automatic grid of your latest published blog posts.

## Editor

- The **block picker is now grouped into categories** — Text, Media,
  Layout, Content, Social proof, Marketing & pricing, Embed & forms, and
  Dynamic — instead of one long list. Searching still shows a single
  ranked list of matches.

## Fixes

- Fixed an editor rendering warning when a previously-selected block was
  restored after a save or reload.
