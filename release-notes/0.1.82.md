## 0.1.82

Two headline additions — edit your whole site with an AI agent, and 22 new content blocks that roughly double the library — plus clickable cards, your own favicon, your logo in Google, finer control over project pages, and a more polished editor.

### New — build with AI agents (API tokens)

CaveCMS now has a real HTTP API with tokens to match. Go to **Settings → API Tokens**, generate a token, and hand it to your favourite AI coding tool — Cursor, Windsurf, Claude Code, OpenAI Codex — or any script. From there your agent edits your live site by prompt: create pages, rewrite blocks, reorder sections, repaint the whole palette — in seconds, with no browser in the loop. The visual builder is still right there for your hands; this is for everything else.

- **Two roles.** Each token is **editor** or **admin** — you choose when you create it.
- **Capped on purpose.** A token can touch your content and branding only. It can **never** reach your user accounts, security settings, secrets, or the updater — even an admin token. So handing one to a tool is safe.
- **Shown once, revocable anytime.** The token appears a single time at creation (we only ever store a hash). Set an optional expiry, and revoke any token in one click — it stops working that second.
- **It just sends a header.** `Authorization: Bearer …` — no cookie, no CSRF dance. That is what makes agent edits fast.

Every install also ships an **AGENTS.md** guide at its root — the file Cursor, Claude Code, Windsurf, and Codex read automatically. After you update, point your AI tool at it and give it a token; it explains exactly how to edit your site (and what it must leave alone). This is separate from the in-editor AI writing partner — that one helps *you* write; this one lets an agent build *for* you.

### New — 22 new content blocks

The block library roughly doubles in size. Every new block is fully CMS-native — insert, edit, restyle, reorder, and clone it like any other, with no database migration:

- **Media** — Carousel (swipeable, autoplay, captions, per-slide links), Before / after comparison slider, Hotspot (an image with tooltip markers), and Marquee (a logo or text ticker).
- **Social proof** — Testimonial carousel, Reviews (a star-rated grid), and Star rating (with half-stars).
- **Marketing & pricing** — Pricing table (with a featured-plan highlight), Price list, and Share buttons (X, LinkedIn, Facebook, email, copy-link).
- **Content & interactive** — Animated headline (rotate or typewriter), Countdown, Flip box, Progress bars, Progress tracker, Timeline, Comparison table, and Toggle.
- **Layout & navigation** — Table of contents and Menu anchor.
- **Embed & dynamic** — Embed (YouTube, Vimeo, Spotify, CodePen, SoundCloud, CodeSandbox), Code (with filename + line numbers), and Posts (an auto-updating grid of your latest blog posts).

Type **/** anywhere on a page to drop one in.

### New — link a whole card

Make an entire column clickable — image, heading, text and all — as a single link, while any button you drop inside it still works on its own. Perfect for project tiles, feature cards, and team members. Set it from the column's edit drawer.

### New — your own favicon

Upload a favicon under **Settings → SEO** and it shows up in browser tabs and bookmarks. Leave it unset and CaveCMS uses its bundled mark, as before.

### New — your logo in Google results

The "Logo for Google" setting is now a proper image upload instead of a paste-a-URL field — it feeds the structured data search engines use to show your logo. If you don't set one, CaveCMS falls back to your header logo automatically.

### New — more control over project pages

Project sections gained presentation options, each optional and defaulting to today's look: hero content alignment, CTA layout and status-label tone; gallery columns; floor-plan fact orientation and alignment; a payment-schedule display for pricing; and card surface / field styling for the inquiry and brochure forms. Tune the look of a project page without touching a line of code.

### New — Help links straight to the docs & Cave Concierge

The **Help** page in your dashboard now links, right at the top, to the complete and always-current CaveCMS documentation and **Cave Concierge** — the AI docs helper. Ask it anything, or browse the latest guides, without leaving your workflow. The built-in quick reference stays exactly where it is: it ships with your install, so it works offline and always matches the version you're running.

### Editor & polish

- The block picker is now grouped into eight categories, so the bigger library stays easy to browse — search still returns a flat, ranked list.
- "Save as block" and the saved-block delete now use CaveCMS's own branded dialogs instead of the browser's plain pop-ups — and a block you save shows up in the **Saved** tab immediately.
- The default welcome page now centres its hero and "Get started" sections for a cleaner first impression on fresh installs.

### Fixes

- Cleared a console hydration warning from the editor's selection chrome when a block is restored right after a save.

Updating applies everything automatically.
