## 0.1.58

Edit your project pages the same way you edit everything else. Until now,
project detail pages were the one part of your site the in-place editor
couldn't touch — you could rewrite any page, post, or section by clicking it
on the live site, but your projects were locked behind a separate form. That's
fixed: project pages are now built from the same editable blocks as the rest
of your site.

### Project pages are now fully editable

Open a project on your live site, click any piece of it — the headline, the
price, an amenity, the map, a photo, a testimonial — and edit it right there.
Every element is its own block, so you can also reorder sections, duplicate
them, or drop in new ones from the block palette, exactly like you already do
on your other pages. The old project editor's separate "Page sections" panel
is gone; you edit a project on the project page itself now.

Nothing about your projects' content, links, or search listing changes — the
listing cards, the project banner, the brochure download, the enquiry form,
and the Google results all keep working as before. They're simply editable
now, element by element.

### Existing projects convert automatically

When you update, every project you already have is converted to the new
editable format in the background — you don't need to do anything, and your
site stays online the whole time. A project only switches to the new format
once its conversion is verified; anything that can't convert cleanly keeps
rendering exactly as it did before and is noted in your server logs, so a
project never breaks mid-update.

### A sharper Location section

The Location section now lays out as two columns — the neighbourhood details
(address and nearby points of interest, each with its own icon beside the
name) on one side, and a large interactive map filling the other — instead of
a single stacked column with the map tucked underneath.
