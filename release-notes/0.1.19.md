## 0.1.19

Quality-of-life: the running CaveCMS version is now always visible.

### Added

**Version chip beside the wordmark in the admin sidebar.** The desktop
sidebar header now renders `v0.1.19` (in muted copper-cream typography)
next to the "CaveCMS" wordmark — operators can confirm at a glance
which version they're on without navigating to Settings → Updates.

The version is bundled at build time from `package.json` (JSON import
inlined by webpack), so it survives the standalone bundle's typical
package.json gaps. Tooltip on hover spells it out for screen readers
and for keyboard navigation.

### Migration impact

None. Pure UI addition.
